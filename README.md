### Instalacja
Żeby odpalić pliki z nagłówkiem `#!/bin/bash` etc musimy nadać uprawnienia. Robimny to za pomoca polecenia `chmod +x test.py` i pozniej test wywołujemy: `./test.py`

**Pamiętać o zmianie adresów ip w plikach**

Pamiętać o zmianie sposobu łączenia się z podania hasła na łączenie się przez klucz SSH. [Poradnik](https://hackmd.io/@olekczubiak/ry911LP7O)
